# Repository Guidance

The Laravel application is located in `src/`.

For any task that reads or changes files under `src/`, first read `src/AGENTS.md` completely and follow its Laravel Boost guidelines. Activate the relevant project skills from `src/.agents/skills/` before working in their domains.

Run Laravel, Composer, Artisan, Pest, Pint, npm, and Vite commands from `src/` unless a command explicitly targets the repository root.
